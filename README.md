📘 Exam Hall Management System
📌 Project Overview

The Exam Hall Management System is a web-based application developed using Python and Streamlit that helps automate the process of allocating students and staff to exam halls.
This project reduces manual effort and ensures accurate seating arrangements for examinations.

The system allows users to upload data, generate seating plans, assign invigilators, and download reports in PDF format.

🔗 Live Website: https://exam-hall-project.streamlit.app/

🎯 Features
📊 Student allocation to exam halls
👩‍🏫 Staff / Invigilator allocation
🪑 Automatic seating arrangement
📂 Upload data using Excel file
📄 Download seating details as PDF
⚡ Fast and simple user interface
💻 Built using Python Streamlit framework
